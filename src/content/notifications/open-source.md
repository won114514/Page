# 网站已开源！

🎉 好消息！本网站的源代码已正式在 GitHub 上开源！

## 开源信息

- **GitHub 仓库**：[won114514/Page](https://github.com/won114514/Page/)
- **开源协议**：MIT License
- **主要技术栈**：Astro、TypeScript、Markdown
- **开源日期**：2026年2月1日

## 项目特点

- **内容优先**：只需在 `content/posts/` 目录中编写 Markdown 文件即可发布文章
- **零配置发布**：运行一条命令即可生成完整的静态网站
- **开发者友好**：项目结构清晰、类型安全、易于扩展
- **极简设计**：白底黑字，良好排版，响应式布局
- **深色/浅色模式**：支持主题切换
- **RSS 订阅**：自动生成 RSS  feeds
- **代码高亮**：集成 Shiki 实现美观的代码高亮

## 贡献指南

欢迎大家为项目贡献力量：

- **提交 Issue**：反馈问题或提出新功能建议
- **提交 PR**：改进代码或添加新功能
- **分享传播**：将项目推荐给更多人
